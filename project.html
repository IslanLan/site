Chemicalsearch

Programa desenvolvido com a finalidade de auxiliar em pesquisas de elementos químicos. Através deste programa é possível com uma rápida pesquisa apenas digitando a sigla do elemento químico desejado: Descobrir a Estabilidade dos Elementos, Consulte a classificação de cada elemento (metais, não-metais, gases nobres, entre outros), Verificar a densidade dos elementos,Verificar o número atômico do elemento pesquisado, Verificar a massa molecular, Observar a configuração eletrônica dos elementos químicos e Verificar quem descobriu o elemento químico pesquisado.
